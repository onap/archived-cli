Linux Command Console

Please follow the instructions given below for installing ONAP CLI.

  1. Download ONAP CLI from here
  2. Unzip into folder /opt/onap/cli
  3. Initialize CLI by running /opt/onap/cli/cli-init.sh
  4. Run ONAP CLI, by typing onap from command line
  5. For more details, refer /opt/onap/cli/README.md
